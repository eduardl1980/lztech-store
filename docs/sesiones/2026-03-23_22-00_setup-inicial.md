## [22:20] Prompt #06 — Productos vacíos desde red
**Pregunta:** `desde localhost veo mis productos y desde network lo veo vacio, por que es?`
**Accion:** Se diagnosticó que Windows Firewall bloqueaba el puerto 3000. Los `fetch` a `/api/*` fallaban silenciosamente desde otros dispositivos de la red. Se creó regla de firewall TCP puerto 3000 para todos los perfiles. Se agregó `allowedDevOrigins: ["192.168.0.10"]` en `next.config.js` para habilitar HMR desde red. API respondiendo 200 desde red confirmado en logs.
